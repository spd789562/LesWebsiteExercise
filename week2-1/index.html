夜市介紹
YES
Introduction

在以前，夜市通常是沒那麼有錢的人在去的地方
大部分東西都很便宜，如衣服、食物等
隨著時代演變，夜市也漸漸變成許多人開業之地
人潮也愈來愈多，甚至成為了觀光景點
根據交通部觀光局「2014 年來臺旅客消費及動向調查」
來台旅客最常去的第一名，就是夜市
到這等不及想知道夜市有什麼了對吧?趕快跟著我一起了解吧!

夜市好好吃
　YES so yaming

夜市中最不缺的就是美食了
從最常看到的蚵仔煎、臭豆腐、碗糕
到傳統的肉燥飯、滷味、乾麵、豆花
還有小朋友最愛的糖葫蘆
真的是應有盡有呢~

夜市好好玩　
YES so happy

夜市不僅僅有很多吃的
能遊玩的東西也很多呢~
像是彈珠檯
看著彈出去的小鋼珠在盤裡撞來撞去，分數夠還能拿獎品呢!
還有射氣球
有的是拿槍、有的是拿飛鏢，我自己最愛玩飛鏢了，就算射不到也玩得很盡興
當然還有最經典的套圈圈拉
套到就是你的!如果目標很後面的話，難度很高呢~
